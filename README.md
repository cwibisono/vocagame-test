# VocaGame Test Automation

This repository contains automated test cases for the **VocaGame** application, built using **Katalon Studio** focuses on validating core UI and functional flows with explicit assertions to ensure that every critical step succeeds or fails as expected.

---

## Project Overview

* **Automation Tool**: Katalon Studio
* **Test Type**: UI & Functional Automation
* **Target**: VocaGame web
* **Focus**:

  * Clear and maintainable test cases
  * Explicit assertions on critical steps
  * Reusable test objects and custom keywords

---

## Repository Structure

```
├── Profiles/               # Execution profiles & test data
├── Keywords/               # Custom reusable keywords
├── Object Repository/      # UI element definitions
├── Test Cases/             # Test case definitions
```

---

## Requirements

Before running the tests, ensure the following are installed:

* Katalon Studio (compatible version)
* Java JDK 11 or newer
* Supported browser drivers (Chrome / Firefox)
* Git
